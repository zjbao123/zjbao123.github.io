JVM垃圾收集器

发表于 2021-10-13 | 分类于 Java

本文主要介绍JVM垃圾回收机制以及实际遇到的内存收缩能力相关的问题以及解决方案。

这里的内存收缩能力主要是指：JVM在向操作系统申请内存使用之后，在进行JVM垃圾回收并且释放一定内存后是否一直占用操作系统的内存。若JVM一直占用，将导致ACS容器平台中的内存利用率较低，很多空余的内存没有得到实际的使用。

前提都是在Xms和Xmx不一致并且相差不接近的情况。
